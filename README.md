# REVIEW-BOT

![Alt text](./assets/logo.png)

## About:
- A simple discord bot. A MEME REVIEWER bot.

## Usage:
- You will need to put your bot token in a `.env` file. See `.env_example`.
- To start the bot, run `npm start`.

- Kill Running Node:
	- command1: ps -e | grep node
	- command2: kill -9 {id}
